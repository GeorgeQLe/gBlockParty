## What we're building

A fully functional pastebin with syntax highlighting, optional expiration, and short shareable URLs — deployed on Vercel with a Neon Postgres backend.

<Callout tone="info">
This tutorial assumes you have Node 20+ and a free Neon account. We'll set up everything else from scratch.
</Callout>

## Stack

- **Next.js 15** — App Router, Server Actions
- **Neon** — Serverless Postgres
- **Drizzle ORM** — Type-safe queries
- **Shiki** — Syntax highlighting at build time

<RepoCard url="https://github.com/gblockparty/pastebin-clone" name="pastebin-clone" description="Starter repo for the Pastebin Clone tutorial" />

## Step 1 — Project setup

Scaffold the Next.js app and wire up Drizzle with the Neon driver.

## Step 2 — Schema and migrations

Define the `pastes` table with columns for content, language, expiration, and a nanoid slug.

## Step 3 — Create paste flow

Server Action that validates input, inserts a row, and redirects to the paste URL.

## Step 4 — Paste detail page

Dynamic route that fetches by slug, highlights with Shiki, and handles 404 / expired states.

## Step 5 — Deploy

Push to Vercel, connect Neon, and verify the production build.
