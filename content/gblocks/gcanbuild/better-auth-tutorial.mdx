## Why Better Auth?

Auth libraries come and go. Better Auth stands out because it owns the session layer without owning your database — you keep Drizzle migrations, it keeps the cookie jar.

<Callout tone="warn">
Better Auth is pre-1.0. Pin your version and check the changelog before upgrading in production.
</Callout>

## What we'll cover

1. Install and configure Better Auth with the Drizzle adapter
2. Email/password signup and login
3. GitHub OAuth provider
4. Protecting server actions and API routes
5. Session management and sign-out
